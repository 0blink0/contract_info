# 合同抽取字段清单（v1）

> 依据：`产品要素-2.xlsx`、`产品运营费率导入模板-1.xlsx`  
> 路径：**A = 模板 Excel 导入**（本期实现）｜**B = 系统页面手录**（本期不做自动填表）  
> 抽取：**规则** = 章节/表格/正则｜**LLM** = 语义理解 + 枚举映射｜**混合** = 规则候选 + LLM 归一化｜**人工** = 合同常无或运营录入

**合同章节缩写**：基本情况｜募集｜成立备案｜申赎｜投资｜费用｜收益分配｜风险｜附件

## 七、实现备注

1. **模板重复列**：`开放日规则`、`止损线` 各出现 2 次，写 Excel 时按表头顺序填同一值或合并为一列逻辑源。  
2. **「内部计算参数」**：业务若仅指费率，则只实现第四节；若有第二套 Excel 模板需补表。  
3. **抽取接口**：`parse` → Document JSON；`extract_product` / `extract_fees` / `extract_lock` / `extract_share_class` → 各子 JSON；`export_xlsx` → 填模板。
