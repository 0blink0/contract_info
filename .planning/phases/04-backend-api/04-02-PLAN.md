```python
IN_PROGRESS = {"parsing", "extracting", "exporting"}

def run_pipeline(file_id: uuid.UUID) -> None: ...
```

**can_run / run 前置：**
- `IN_PROGRESS` → 抛 `PipelineBusyError`（HTTP 409）
- `exported` → `PipelineCompleteError`（409）
- 其余按 RESEARCH 表续跑

**run_pipeline：**
- `pending` / `failed` → `parse_file_id` → `persist_extract` → `persist_export`
- `parsed` / `extraction_failed` → extract → export
- `extracted` / `export_failed` → `persist_export` only
- 顶层 except：确保 DB 有 error（各 service 已写 failed 状态）

供 BackgroundTasks 与测试直接调用。
