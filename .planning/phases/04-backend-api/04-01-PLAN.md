**upload_service.persist_upload(content: bytes, filename: str) -> uuid.UUID：**
- 校验扩展名 `.docx`（忽略大小写），否则 ValueError
- 生成 file_id，目录 `uploads/{file_id}/`，写入文件

**parse_service：** 抽出 `parse_file_id(file_id: uuid.UUID) -> uuid.UUID`：
- 从 DB 读 `storage_path`，`status=parsing`，调用 `parse_docx`，写 `parse_json`/`outline_preview`，成功 `parsed`，失败 `failed` + error_message
- `persist_parse(path)` 保持 CLI 兼容：可先 copy+parse 一条路径，或 upload+parse_file_id 组合（实现者择一，CLI 测试仍通过）
  </action>
