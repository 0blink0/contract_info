## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | 极小元素内边距、标签紧凑间距 |
| sm | 8px | 表格工具条按钮间距、次级信息组间距 |
| md | 16px | 卡片内常规区块间距、表格上方操作区 |
| lg | 24px | 页面主容器内边距（`page-shell`） |
| xl | 32px | 页面大区块分隔 |
| 2xl | 48px | 模块级垂直留白 |
| 3xl | 64px | 全页级分段留白（仅大屏） |

Exceptions: 行高可用 40px/44px（菜单项触达与可读性；来源: `AppLayout.vue` 现有模式）

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 14px | 400 | 1.5 |
| Label | 16px | 600 | 1.5 |
| Heading | 22px | 600 | 1.2 |
| Display | 28px | 600 | 1.2 |

约束说明：
- 字号固定 4 档：14 / 16 / 22 / 28。
- 字重固定 2 档：400（正文）+ 600（标题/标签）。
- 与现有页面标题、描述和导航风格保持一致（来源: `app.css`, `AppLayout.vue`, `FileListView.vue`）。

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#f4f6fb` | 页面背景、主工作区底色（来源: `--app-bg`） |
| Secondary (30%) | `#ffffff` | 卡片、表格容器、弹窗主体（来源: `--app-surface`） |
| Accent (10%) | `#3b82f6` | 主要按钮、菜单激活态、分页当前页、高亮交互 |
| Destructive | `#f56c6c` | 删除按钮、删除确认主动作、危险提示 |

Accent reserved for: `搜索/刷新` 触发按钮、菜单「知识库配置」激活态、分页当前页指示、表格内关键可操作入口；不得用于纯说明文字。

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | 搜索案例 |
| Empty state heading | 暂无知识库案例 |
| Empty state body | 当前筛选条件下没有匹配记录。请调整字段名关键词，或先在「字段 B」页面存入案例。 |
| Error state | 知识库列表加载失败，请点击“刷新”重试；若仍失败，请检查本地模型/接口状态。 |
| Destructive confirmation | 删除案例：确定删除该知识库案例？删除后不可恢复。 |

补充交互文案约束（来源: `21-CONTEXT.md` D-05/D-06）：
- 删除成功：`已删除 1 条案例`。
- 删除失败：`删除失败：{后端错误摘要}`。
- 删除后保持当前筛选词和当前页，不重置上下文。

## Interaction Contract (Phase 21 Scope)

| Requirement | Contract |
|-------------|----------|
| KB-UI-01 菜单入口 | 在左侧导航新增 `知识库配置`，位置固定在 `文件列表` 下方、`系统设置` 上方；沿用 `default-active + router`，不影响 `文件详情` 子菜单展开。 |
| KB-UI-02 列表展示 | 列表列固定为：字段名 / 字段值 / 原文摘录 / 来源合同 / 入库时间 / 操作；采用服务端分页查询路径（首屏即分页请求）。 |
| KB-UI-03 字段名过滤 | 搜索范围仅 `field_name`；输入防抖即时查询（300ms 默认），过滤词变更后强制回到第 1 页。 |
| KB-UI-04 删除闭环 | 操作列提供删除；先二次确认，再调用删除 API；成功后在当前查询上下文重新拉取列表，取消时列表不变。 |

性能与一致性约束（来源: `21-CONTEXT.md` D-03/D-04）：
- 禁止前端全量拉取后本地过滤。
- 快速输入时必须避免竞态覆盖（请求取消或序列号保护）。
