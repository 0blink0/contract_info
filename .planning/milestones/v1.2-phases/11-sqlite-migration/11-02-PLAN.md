## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-11-03 | Information Disclosure | sa.JSON on SQLite stores data as TEXT | accept | JSON TEXT storage is standard SQLite practice; no encryption needed for this app's data |
| T-11-04 | Tampering | sa.Uuid stored as CHAR(32) hex — no hyphen validation | accept | SQLAlchemy enforces uuid.UUID type at Python level with as_uuid=True; invalid UUIDs cannot enter via ORM |
| T-11-05 | Tampering | server_default CURRENT_TIMESTAMP on SQLite has no timezone | accept | App uses client-side func.now() in ORM (has timezone); server_default is DDL fallback only; acceptable for desktop app |
| T-11-SC | Tampering | npm/pip/cargo installs | accept | No new packages installed in this phase |
</threat_model>
