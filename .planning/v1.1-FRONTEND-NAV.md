# v1.1 Web 前端导航重构（2026-05-26）

纳入 **v1.1 交付范围**（Phase 10 运营面增强，非新 Phase）。

## 变更

| 项 | 说明 |
|----|------|
| 路由 | `vue-router`：`/upload` 上传解析、`/jobs` 历史列表、`/jobs/:id` 文件详情 |
| 上传页 | 拖拽上传、步骤条、开始/重试、完成后「查看结果」 |
| 文件列表 | 表格展示历史任务，查看详情 / 删除（处理中不可删） |
| 文件详情 | 原右侧详情能力：下载五表、校验、Path B、导出预览 |
| 样式 | 深色侧栏 + 卡片式内容区 |
| Bugfix | `ValidationPanel` / `PathBPanel` 在 `jobId` 变化时重置缓存；详情子组件 `:key` 强制重建 |

## 部署

```bash
docker compose up -d --build web
# 或本地
cd frontend && npm install && npm run build
```

Nginx 已配置 `try_files … /index.html`，支持前端路由刷新。
